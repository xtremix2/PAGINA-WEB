Nosotros





De parte del equipo de InclusiWeb:


Queremos expresar nuestro sincero agradecimiento a todas las personas y entidades que han sido parte fundamental en la creación y desarrollo de esta página web. Sus contribuciones, apoyo y colaboración han hecho posible que este proyecto sea una realidad.


Nosotros:

El equipo de InclusiWeb, compuesto por cuatro estudiantes del Instituto Tecnológico de Cerro Azul, ha abrazado la misión de crear una plataforma web inclusiva que promueve la diversidad digital y la igualdad de oportunidades en línea. Este grupo de talentosos jóvenes se ha unido con un objetivo claro: utilizar la tecnología como una herramienta para eliminar barreras y fomentar un espacio digital verdaderamente accesible para todos. Cada miembro del equipo aporta sus habilidades y pasiones únicas a InclusiWeb:

1. Octavio:

Gestiona y crea contenido relevante y atractivo para plataformas digitales, asegurando precisión y coherencia.

2. Luis:

Crea y mantiene la estructura y funcionalidad de sitios web utilizando lenguajes de programación.

3. César:

Crea elementos visuales efectivos, como logotipos y gráficos, para transmitir mensajes y mantener la coherencia visual de la marca.

4. Hugo:

Enfocado en la estética y usabilidad, diseña la apariencia visual y la disposición de elementos en sitios web antes de la implementación técnica.

El equipo de InclusiWeb, del Instituto Tecnológico de Cerro Azul, trabaja incansablemente para garantizar que InclusiWeb, sea un recurso invaluable para personas de todas las edades, orígenes y habilidades. Su dedicación a la inclusión digital es evidente en cada aspecto de su trabajo, desde el diseño accesible hasta el contenido educativo que ofrecen. Con InclusiWeb, estos estudiantes están contribuyendo de manera significativa a un mundo digital más inclusivo y equitativo, y están demostrando el poder de la tecnología para transformar vidas y comunidades.

Recursos y Herramientas:

No podríamos haber logrado esto sin el apoyo de las siguientes herramientas y recursos:
Visual Studio Code: Es una excelente elección para la creación de una página web como InclusiWeb,
Lenguaje HTML, CSS y JavaScript: VS Code es un potente editor de texto que brinda soporte para la creación y edición de código HTML, CSS y JavaScript, los lenguajes fundamentales para desarrollar una página web.
